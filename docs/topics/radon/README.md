**Talk slot:** 2 (after Legend State, partner)

## Audience takeaway

Radon IDE is worth adopting for day-to-day RN work; `rnrepo` is a practical add-on that saves real build time in EAS — especially on Android.

## What to cover

### Radon IDE (main focus)

- Advantages over a generic editor for React Native
- Faster navigation across **JS + native** layers
- RN-aware tooling, debugging, project introspection tuned for mobile
- Frame for **business apps** with non-trivial native surface — not a toy IDE demo

### rnrepo (short mention)

- Same ecosystem; monorepo / workspace build optimization
- Lead with measurable win: **shorter Android builds in EAS** (and similar CI)
- One slide or cached timing screenshot — avoid cold EAS on stage
